Lists

Ordered List

The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

Code Example:

        <ol>
          <li>One</li>
          <li>Two</li>
          <li>Three</li>
          <li>Four</li>
          <li>Five</li>
        </ol>

Unordered List

The <ul> tag defines an unordered (bulleted) list.

  • One
  • Two
  • Three
  • Four
  • Five

Code Example:

        <ul>
          <li>One</li>
          <li>Two</li>
          <li>Three</li>
          <li>Four</li>
          <li>Five</li>
        </ul>
