Media

Audio

The element <audio> represents a sound or audio stream.

Code Example:

<audio controls>
    <source src="/assets/powerful-beat-121791.mp3" type="audio/mpeg">
</audio>

Video

The element <video> tag is used to embed video content in a document, such as a movie clip or other video streams.

Code Example:

<video width="420" height="315" controls>
    <source src="/assets/video.mp4" type="video/mp4">
</video>

Youtube iframe

The element <iframe> represents a nested browsing context.

Code Example:

<iframe width="420" height="315" src="https://www.youtube.com/embed/TwIvUbOhcKE"></iframe>
