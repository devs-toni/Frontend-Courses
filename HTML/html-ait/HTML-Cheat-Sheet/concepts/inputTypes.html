Input Types

Button

The element <input type="button"> defines a clickable button (mostly used with a JavaScript to activate a script).

Code Example:

<input type="button" value="Button" onclick="show()">

CheckBox

The element <input type="checkbox"> defines a checkbox that is shown as a square box that is ticked (checked) when activated.

Code Example:

<input type="checkbox" id="checkboxId" name="checkboxName" value="example">

Color

The element <input type="color"> defines a color picker.

Code Example:

<input type="color" id="colorId" name="colorName" value="#000000">

Date

The element <input type="date"> defines a date picker.

Code Example:

<input type="date" id="dateId" name="dateName">

Month

The element <input type="month"> defines a month and year control.

Code Example:

<input type="month" id="monthId" name="monthName">

Email

The element <input type="email"> defines a field for an e-mail address.defines a field for an e-mail address.

Code Example:

<input type="email" id="emailId" name="emailName">

File

The element <input type="email"> defines a file-select field and a "Browse" button for file uploads.

Code Example:

<input type="file" id="fileId" name="fileName">

Hidden

The element <input type="hidden"> defines a hidden input field.

Code Example:

<input type="hidden" id="hiddenId" name="hiddenName" value="example">

Image

The element <input type="image"> defines an image as a submit button.

Code Example:

<input type="image" src="assets/pinguino.png" alt="Submit" width="265" height="150">

Number

The element <input type="number"> defines a field for entering a number.

Code Example:

<input type="number" id="numberId" name="numberName min="1" max="20>

Password

The element <input type="password"> defines a password field (characters are masked).

Code Example:

<input type="password" id="passwordId" name="passwordName">

Radio Button Group

The element <input type="radio"> defines a radio button.

Radio Button Group

Code Example:

<input type="radio" id="radioId" name="radioName" value="example">

Range

The element <input type="range"> defines a control for entering a number whose exact value is not important (like a slider control).

Code Example:

<input type="range" id="rangeId" name="rangeName" min="0" max="10">

Reset

The element <input type="reset"> defines a reset button which resets all form values to its initial values

Code Example:

<input type="reset">

Search

The element <input type="search"> defines a text field for entering a search string.

Code Example:

<input type="search" id="searchId" name="searchName">

Submit

The element <input type="submit"> defines a submit button which submits all form values to a form-handler.

Code Example:

<input type="submit">

Tel

The element <input type="tel"> defines a field for entering a telephone number.

Code Example:

<input type="tel" id="telId" name="telName" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}">

Text

The element <input type="text"> defines a single-line text field.

Code Example:

<input type="text" id="textId" name="textName">

Time

The element <input type="time"> defines a control for entering a time (no time zone).

Code Example:

<input type="time" id="timeId" name="timeName">

Url

The element <input type="url"> defines a field for entering a URL.

Code Example:

<input type="url" id="urlId" name="urlName">

Week

The element <input type="week"> defines a week and year control (no time zone).

Code Example:

<input type="week" id="weekId" name="weekName">
