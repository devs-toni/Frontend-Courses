Images

The element <img> represents an image.

alt attribute defines an alternative text description of the image.

Code Example:

<img src="ejemplo.png" alt="this is what appears when there is an error if the browser can't open the image">

With a fixed height and width.

Code Example:

<img src="./assets/pinguino.png" height="150" width="265" alt="image">

can turn the image into a link by adding the image instead of text when adding a link.

Code Example:

<a href="index.html" target = "_self"> <img src="/assets/pinguino.png" height="150" width="265" alt = "Image as link"> </a>
