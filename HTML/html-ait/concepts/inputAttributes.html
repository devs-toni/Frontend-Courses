Input attributes

Value

The element value="example" specifies the default value of an <input> element.

Code Example:

<input type="text" id="valueId" name="valueName" value="Pablo">

Readonly

The element readonly, when present, it specifies that the <input> element should be disabled.

Code Example:

<input type="text" id="readonlyId" name="readonlyName" value="Spain" readonly>

Disabled

The element disabled, when present, it specifies that an input field is read-only.

Code Example:

<input type="text" id="disabledId" name="disabledName" value="Pablo" disabled>

Size

The size="6" attribute specifies the visible width, in characters, of an <input> element.

Code Example:

<input type="text" id="sizeId" name="sizeName" size="6>

Maxlength

The maxlength="6" attribute specifies the maximum number of characters allowed in the <input> element.

Code Example:

<input type="text" id="maxlengthId" name="maxlengthName" maxlength="6>

Min and Max

The min="1" and max="3" attributes specify the minimum and maximum values for an <input> element.

Code Example:

<input type="number" id="minMaxId" name="minMaxName" min="1" max="10">

Min and Max

The multiple attribute specifies that the user is allowed to enter more than one value in the <input> element. The multiple attribute works with the following input types: email (separated with commas), and file (CTRL or SHIFT while selecting).

Code Example:

<input type="email" id="multipleId" name="multipleName" multiple>

Pattern

The pattern attribute specifies a regular expression that the <input> element's value is checked against on form submission.

Code Example:

<input type="text" id="patternId" name="patternName" pattern="[A-Za-z]{3}">

Placeholder

The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format).

Code Example:

<input type="text" id="placeholderId" name="placeholderName" placeholder="Country Code">

Required

The required attribute when present, it specifies that an input field must be filled out before submitting the form.

Code Example:

<input type="text" id="requiredId" name="requiredName" required>

Step

The step attribute specifies the interval between legal numbers in an <input> element

Code Example:

<input type="text" id="stepId" name="stepName" step="3">

Autofocus

The autofocus attribute when present, it specifies that an <input> element should automatically get focus when the page loads.

Code Example:

<input type="text" id="autofocusId" name="autofocusName" autofocus>

Height and Width

The height and width attributes specifies the height and the width of the <input> element.

Code Example:

<input type="image" id="heightWidthId" name="heightWidthName" alt="image" height="100" width="300" >

List

The list attribute refers to a <datalist> element that contains pre-defined options for an <input> element.

Code Example:

<input list="browsers">
<datalist id="browser">
    <option value="Edge">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
</datalist>

Form

The form attribute specifies the form the <input> element belongs to.



Code Example:

<form action="/action_page.php" id="form1">
    <label for="fname">First name:</label>
    <input type="text" id="fname" name="fname"><br><br>
    <input type="submit" value="Submit">
</form>

<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname" form="form1">
